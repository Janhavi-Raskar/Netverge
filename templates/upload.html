How to Upload Logs

  1. Click the Choose File button below.
  2. Select a .log file in Snort format (see sample on Home page).
  3. Click Upload to process and analyze the file.
Only .log files are accepted. Each file should contain Snort alert entries with alert message, priority, timestamp, and IPs.
